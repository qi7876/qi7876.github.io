Tmux 是一个功能强大的终端复用器（Terminal Multiplexer），学会它能极大地提高你的工作效率。

### 核心理念：前缀键 (Prefix Key)

Tmux 的所有快捷键都需要先按下一个**前缀键**，然后再按功能键。默认的前缀键是 `Ctrl + b`。

**操作流程**：
1.  按住 `Ctrl`，再按 `b`。
2.  **松开**这两个键。
3.  再按下一个功能键（例如 `c`、`d`、`%` 等）。

在下面的所有命令中，`[Prefix]` 就代表 `Ctrl + b`。

---

### 一、会话管理 (Session Management)

会话是 Tmux 的最高层级，你可以拥有多个独立的会话，每个会话里可以有多个窗口和窗格。这些命令通常在**终端的命令行**中直接输入，而不是在 Tmux 内部。

| 命令                                  | 功能描述                       |
| :------------------------------------ | :----------------------------- |
| `tmux new -s <session_name>`          | 创建一个指定名称的新会话。     |
| `tmux ls`                             | 列出所有正在运行的 Tmux 会话。 |
| `tmux attach -t <session_name>`       | 重新连接到指定的会话。         |
| `tmux kill-session -t <session_name>` | 关闭指定的会话。               |
| `tmux kill-server`                    | 关闭所有会话和 Tmux 服务。     |

**在 Tmux 内部操作会话：**

| 快捷键           | 功能描述                                           |
| :--------------- | :------------------------------------------------- |
| `[Prefix]` + `d` | **分离 (Detach)** 当前会话，程序会继续在后台运行。 |
| `[Prefix]` + `s` | 以列表形式**选择 (Switch)** 其他会话。             |
| `[Prefix]` + `$` | **重命名 (Rename)** 当前会话。                     |

---

### 二、窗口管理 (Window Management)

窗口类似于浏览器的标签页，每个会话可以包含多个窗口，让你在不同任务间切换。

| 快捷键               | 功能描述                                            |
| :------------------- | :-------------------------------------------------- |
| `[Prefix]` + `c`     | **创建 (Create)** 一个新窗口。                      |
| `[Prefix]` + `w`     | 以列表形式**选择 (Switch)** 窗口。                  |
| `[Prefix]` + `p`     | 切换到**上一个 (Previous)** 窗口。                  |
| `[Prefix]` + `n`     | 切换到**下一个 (Next)** 窗口。                      |
| `[Prefix]` + `[0-9]` | 直接切换到指定编号的窗口（例如 `[Prefix]` + `1`）。 |
| `[Prefix]` + `,`     | **重命名 (Rename)** 当前窗口。                      |
| `[Prefix]` + `&`     | 关闭当前窗口（会提示确认）。                        |

---

### 三、窗格管理 (Pane Management)

窗格是 Tmux 的精髓，它允许你在一个窗口内分割出多个独立的终端区域。

| 快捷键                           | 功能描述                                                    |
| :------------------------------- | :---------------------------------------------------------- |
| `[Prefix]` + `"`                 | **水平分割 (Horizontal Split)**，将当前窗格上下拆分。       |
| `[Prefix]` + `%`                 | **垂直分割 (Vertical Split)**，将当前窗格左右拆分。         |
| `[Prefix]` + `[方向键]`          | 根据箭头方向在不同窗格间**切换**。                          |
| `[Prefix]` + `o`                 | 按顺序**循环切换**到下一个窗格。                            |
| `[Prefix]` + `x`                 | 关闭当前窗格（会提示确认）。                                |
| `[Prefix]` + `z`                 | **最大化/还原 (Zoom)** 当前窗格。再按一次可恢复原状。       |
| `[Prefix]` + ` ` (空格键)        | 在预设的几种布局之间切换。                                  |
| `[Prefix]` + `Ctrl` + `[方向键]` | 按住 `Prefix` 和 `Ctrl`，再按方向键，可以**调整窗格大小**。 |
| `[Prefix]` + `q`                 | 显示窗格编号，然后按数字键可快速切换到对应窗格。            |

---

### 四、其他实用命令

| 快捷键           | 功能描述                                                     |
| :--------------- | :----------------------------------------------------------- |
| `[Prefix]` + `t` | 在当前窗格显示一个**时钟**。                                 |
| `[Prefix]` + `[` | 进入**复制模式 (Copy Mode)**。                               |
| `[Prefix]` + `]` | **粘贴 (Paste)** 在复制模式中复制的内容。                    |
| `[Prefix]` + `:` | 进入**命令模式**，可以输入 Tmux 命令（例如 `setw synchronize-panes`）。 |

#### 复制模式详解 (非常有用)

进入复制模式后，你可以像在 Vim 编辑器中一样移动光标和复制文本。
1.  按 `[Prefix]` + `[` 进入复制模式。
2.  使用方向键或 `h, j, k, l` 移动光标。
3.  移动到想复制的文本开头，按 `Space` (空格键) 开始选择。
4.  移动光标到结尾。
5.  按 `Enter` (回车键) 完成复制并退出复制模式。
6.  按 `[Prefix]` + `]` 粘贴内容。

### 高级技巧：自定义

如果你觉得 `Ctrl + b` 不顺手，可以修改它。在你的用户主目录下创建 `~/.tmux.conf` 文件，并添加以下内容，可以将前缀键改为 `Ctrl + a`（这是 `screen` 命令的默认前缀，很多老用户更习惯）：

```tmux
# 设置前缀键为 Ctrl + a
unbind C-b
set-option -g prefix C-a
bind-key C-a send-prefix
```

Tmux 的强大之处在于其高度可定制性，你可以通过 `.tmux.conf` 文件定义自己喜欢的快捷键和外观。
